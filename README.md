# My Browser Tweaks

This repository contains various browser enhancements, tools, and customisations I use for my personal workflow. Feel free to use or fork!

### Contents

* **Browser Extensions:** Full extensions for Chromium-based browsers (e.g., Opera/Chrome).
* **Userscripts:** JavaScript files for use with userscript managers like **Tampermonkey**.
* **Userstyles:** CSS files for website customisation via style managers like **Stylus**.
